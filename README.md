# Flake Config

My nix flake configuration for my personal setup, including a custom Wayland compositor and a tailored shell environment.

## 🚀 Main Components

### [Niri WM](https://github.com/YaLTeR/niri)

Wayland wm lightweight with scrollable workspaces, dynamic tiling, and a focus on simplicity and performance. It provides a clean and efficient environment for managing windows and workspaces.

### [Noctalia-shell](https://github.com/noctalia-dev/noctalia-shell)

A custom shell environment built on top of Quickshell
